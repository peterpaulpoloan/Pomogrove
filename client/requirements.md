## Packages
framer-motion | Page transitions and detailed animations for the tree and UI elements
recharts | Visualizing study statistics and streak data
date-fns | Formatting dates for logs and history
lucide-react | Iconography for the UI (already in base stack, just confirming usage)

## Notes
- Wouter <Link> components render as anchor tags, so no nested <a> tags.
- Replit Auth is used for authentication.
- API requests use `credentials: "include"` for session cookies.
- Tree stage logic: sapling (lvl 1-5), juvenile (lvl 6-15), adult (lvl 16+).
